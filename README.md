### Сборка программы 

```bash
git clone https://github.com/4evrinspace/ast_compare
cd /ast_compare
mkdir -p build && cd build
cmake ..
make .

# Запуск интерактивного CLI
./ast_compare
```

Также параметры можно передавать напрямую при запуске.


Запуск без параметров начнет работу программу в интерактивном режиме. 
CLI покажет меню. Используйте опцию 1 для добавления файлов, опцию 3 для
проверки всех пар, опцию 5 для проверки внешнего файла по базе данных.

Требования: CMake 3.13+, компилятор C++17, библиотеки разработки LLVM/Clang
(протестировано с LLVM 21 на macOS).
